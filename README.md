# feb-2025-avasjcript-events-and-basic-interactivity

# JavaScript Concepts Demonstration

This project is a simple interactive webpage that demonstrates the following JavaScript concepts:

- **Event Listeners**: Handle button clicks and form submission.
- **Form Validation**: Ensure the user provides required inputs before processing the form.
- **Interactive Elements**: Dynamically add content to the page.

## 🗂️ File Structure

```
/project-folder
├── index.html      # Main HTML file
├── style.css       # External CSS styles
└── script.js       # External JavaScript logic
```

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Fill in the form and interact with the button to see the results.

## 🧪 Features

- Displays an error message if form inputs are left empty.
- Adds new content to the page when a button is clicked.
- Disables the button after the first click.
